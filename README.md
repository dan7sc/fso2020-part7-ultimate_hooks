## Available Scripts

In the project directory, to start the client, you can run:

### `npm start`
or
### `yarn start`

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Start the server

### `npm run server`

Run the server at [http://localhost:3005](http://localhost:3005).<br />

Make requests to paths `/notes` and `/persons`
